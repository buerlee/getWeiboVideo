# getWeiboVideo
一键非自动下载微博视频
# 依赖
python 3.7.2

pip 19.0.2

# 安装

```
pip3 install -r requirements.txt  
```

# 食用

1.设置cookie

2.设置你要下载的weibourl

3.设置你要下载存为的文件名字

4.执行脚本
```
python3 getWeiboVideo.py
```
